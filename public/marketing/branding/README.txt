Cin7 Core Toolbox brand asset pack

Use:
- icon.svg: browser favicon / tiny UI mark
- favicon.ico: traditional browser favicon
- favicon-16.png, favicon-32.png, favicon-48.png: explicit favicon sizes
- apple-touch-icon.png: iOS home-screen icon
- icon-192.png, icon-512.png: PWA manifest icons
- logo-mark.svg / PNGs: app mark
- logo-horizontal-dark.svg: white text for dark backgrounds
- logo-horizontal-light.svg: dark text for light backgrounds

All SVG and PNG files have real transparency where appropriate.
